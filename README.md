# Sistema Estratégico de Gestão – Design de Interiores

Projeto desenvolvido por Talita Freitas.

## 📌 Objetivo

Criar um sistema simples e funcional para gestão financeira de projetos de Design de Interiores, permitindo maior controle estratégico dos honorários, custos e lucratividade.

## 🚀 Funcionalidades

- Cadastro de projetos
- Cálculo automático de honorários
- Controle de custo estimado vs custo real
- Dashboard com indicadores financeiros
- Gráfico de lucro mensal
- Exclusão de projetos
- Geração de relatório executivo em PDF

## 🛠 Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Matplotlib
- ReportLab

## 📊 Aplicação

Sistema voltado para profissionais autônomos e pequenos escritórios de Design que desejam profissionalizar sua gestão financeira.
